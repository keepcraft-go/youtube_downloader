# YouTube 다운로더 - 간단 사용 가이드

## 📌 처음 사용하시는 분들을 위한 가이드

### 1단계: 프로그램 실행
- `YouTube_Downloader.exe` 파일을 더블클릭하세요
- (Mac 사용자: `YouTube Downloader.app` 실행)

### 2단계: YouTube URL 복사
1. 다운로드하고 싶은 YouTube 비디오로 이동
2. 주소창의 URL을 복사 (Ctrl+C 또는 Cmd+C)
   - 예시: `https://www.youtube.com/watch?v=xxxxx`

### 3단계: URL 붙여넣기
- 프로그램의 "YouTube URL" 입력창에 붙여넣기 (Ctrl+V 또는 Cmd+V)

### 4단계: 다운로드 옵션 선택
다음 중 하나를 선택하세요:

#### 🎬 비디오 + 오디오 (권장)
- YouTube 영상 편집에 가장 적합
- 비디오 파일 (.mp4)과 오디오 파일 (.mp3)을 모두 받습니다
- 영상 편집 시 비디오와 오디오를 각각 사용할 수 있습니다

#### 🎥 비디오만
- 비디오 파일 (.mp4)만 받습니다
- 오디오가 포함된 완성된 비디오 파일

#### 🎵 오디오만 (MP3)
- 음악이나 음성만 필요할 때 선택
- MP3 파일로 저장됩니다

### 5단계: 저장 위치 선택 (선택사항)
- 기본적으로 "다운로드" 폴더에 저장됩니다
- 다른 곳에 저장하고 싶다면 "찾아보기" 버튼을 클릭하세요

### 6단계: 다운로드 시작
- "다운로드 시작" 버튼을 클릭하세요
- 진행 상황을 확인하세요
- 완료 메시지가 나타나면 끝!

## 🎓 YouTube 영상 제작자를 위한 팁

### 영상 편집용 다운로드
1. **"비디오 + 오디오" 옵션 선택**을 권장합니다
2. 다운로드되는 파일:
   - `[제목].mp4` - 비디오 파일
   - `[제목]_audio.mp3` - 오디오 파일

### 왜 두 개 파일을 받나요?
- **비디오 파일**: 영상 편집 시 영상 트랙으로 사용
- **오디오 파일**: 
  - 배경음악으로 사용
  - 음성 편집이 필요할 때
  - 다른 영상에 오디오만 추가할 때

### 편집 프로그램에서 사용하기
1. **프리미어 프로 / 파이널 컷 / 다빈치 리졸브**:
   - 비디오 파일과 오디오 파일을 각각 타임라인에 드래그
   - 필요에 따라 분리하여 편집

2. **간단한 편집 (윈도우 동영상 편집기 등)**:
   - 비디오 파일만 사용하거나
   - 오디오 파일을 배경음으로 추가

## ❓ 자주 묻는 질문

### Q: FFmpeg 오류가 나타나요
**A:** 오디오 추출 기능을 사용하려면 FFmpeg가 필요합니다.

**Windows 사용자:**
1. https://ffmpeg.org/download.html 접속
2. Windows 버전 다운로드
3. 압축 해제 후 설치
4. 또는 비디오만 다운로드 옵션 사용

**Mac 사용자:**
```bash
brew install ffmpeg
```

### Q: 다운로드가 안 돼요
- 인터넷 연결을 확인하세요
- YouTube URL이 올바른지 확인하세요
- 비디오가 비공개 설정인지 확인하세요
- 지역 제한이 있는 영상일 수 있습니다

### Q: 다운로드가 너무 느려요
- 인터넷 속도에 따라 다릅니다
- 고화질 비디오는 시간이 더 걸립니다
- YouTube 서버 상태에 따라 속도가 달라질 수 있습니다

### Q: 파일을 찾을 수 없어요
- 프로그램 하단의 로그를 확인하세요
- 기본 저장 위치: C:\Users\[사용자명]\Downloads (Windows)
- 또는 /Users/[사용자명]/Downloads (Mac)

### Q: 저작권 문제는 없나요?
- 개인적인 학습, 연구, 백업 목적으로만 사용하세요
- 다운로드한 콘텐츠를 재업로드하거나 상업적으로 사용하지 마세요
- YouTube의 서비스 약관을 준수하세요

## 💡 사용 예시

### 예시 1: 튜토리얼 영상 백업
```
URL: https://www.youtube.com/watch?v=example123
옵션: 비디오만
→ 완성된 영상을 그대로 저장
```

### 예시 2: 음악 추출
```
URL: https://www.youtube.com/watch?v=music456
옵션: 오디오만 (MP3)
→ 음악 파일만 MP3로 저장
```

### 예시 3: 영상 편집용
```
URL: https://www.youtube.com/watch?v=edit789
옵션: 비디오 + 오디오
→ 비디오와 오디오를 각각 받아서 편집에 활용
```
